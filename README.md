# 📚 Java Learning Playground

Este repositorio fue creado con el objetivo de **aprender y practicar conceptos fundamentales de Java** mediante ejemplos simples y experimentos de código.

La idea principal es tener un espacio donde se puedan probar distintas características del lenguaje sin la complejidad de un proyecto grande.

## 🎯 Objetivo

El propósito de este repositorio es explorar y comprender diferentes funcionalidades del lenguaje **Java**, implementando pequeños ejemplos prácticos para cada concepto.

Este proyecto funciona como:

* 📖 Apuntes de estudio
* 🧪 Laboratorio de pruebas
* 💻 Ejemplos de código reutilizables

## 🧩 Contenido del proyecto

Dentro del repositorio se pueden encontrar ejemplos de:

* **Enums**
* **Anotaciones (Annotations)**
* **Funciones y métodos**
* **Clases y objetos**
* **Interfaces**
* **Colecciones**
* **Principios básicos de programación orientada a objetos**
* Otros experimentos y pruebas relacionadas con Java.

Cada sección contiene ejemplos simples diseñados para **entender el concepto de forma clara**.

## 📂 Estructura (Ejemplo)

```
src/
 ├── enums/
 ├── annotations/
 ├── functions/
 ├── oop/
 └── experiments/
```

Cada carpeta contiene ejemplos enfocados en un concepto específico del lenguaje.

## 🚀 Propósito educativo

Este repositorio **no está pensado como un proyecto de producción**, sino como un espacio de aprendizaje donde se prueban ideas, se experimenta con el lenguaje y se documentan conceptos importantes.

## 🛠 Tecnologías

* Java

## 📌 Nota

El código aquí presente puede evolucionar constantemente a medida que se descubren nuevos conceptos o se realizan más experimentos con el lenguaje.
